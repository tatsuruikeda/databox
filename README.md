### Rコードについて

講義内容を再現できるRコードです．

Rファイルの他にipynbファイルがあり，ipynbファイルはGoogle Colaboratoryで動きます．
説明用のpptxファイルがあります．
**注意！マイドライブ内のR_hokudai2024.ipynbを開く前に，「接続」のランタイムのタイプを「R」にしておく必要があります**
